오늘은 반응형 추가와 .bar를 추가했다. 원래는 aside가 오른쪽에 붙어있었는데, 불필요 할 것 같아서 일단 숨겨두고,

.bar 엘리먼트 아이콘이 나온다. 그 아이콘을 누르면 숨어있던 aside가 나온다. 메뉴도 줄기전에 2번째 메뉴를 눌러둔 상태라면
줄어들었을때 aside 메뉴도 2번째가 active 되어있게 만들었다.
